== Description ==

Parsi Virastar helps to write Persian a little more correctly.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/block-editor-virastar` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
